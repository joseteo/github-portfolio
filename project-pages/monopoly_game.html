Monopoly Game
Experience the thrill of Monopoly in a digital realm. Dive into the world of strategic property trading and domination.
Project Details / Background

The Monopoly Game project brings the iconic board game to the digital world, offering an immersive and entertaining experience. Developed using Java and Ruby, with separate branches for each language, and following the Model-View-Controller (MVC) design pattern, this project explores the fusion of classic gameplay with modern technology.

Key Features:

  • Java-based and Ruby-based implementation adhering to MVC architecture.
  • Separate branches for Java and Ruby development.
  • Interactive 2D graphical user interface (GUI) for an engaging user experience that brings the game to life.
  • Efficient turn management, player actions, and game logic.
  • Strategic gameplay mirroring the classic Monopoly board game.
